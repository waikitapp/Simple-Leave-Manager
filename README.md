== Description ==
A simple plugin to manage employee leave requests.

Features
- Submit leave request from frontend
- Admin can approve or reject requests
- Simple and lightweight

== Installation ==
1. Upload the plugin to wp-contentplugins
2. Activate the plugin
3. Use shortcode [leave_form] to display the form
